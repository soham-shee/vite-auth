
## Vite-Authentication Boilerplate

Easy-to-use Vite-Authentication Boilerplate
- Authentication features - Email, Password
- Appwrite Integration

  
## Run Locally

Clone the project

```bash
  git clone https://github.com/soham-shee/vite-auth.git
```

Configuration

```bash
  cd vite-auth
  npm install
```

Start the server

```bash
  npm run dev
```
